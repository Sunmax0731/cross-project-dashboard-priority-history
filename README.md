# cross-project-dashboard-priority-history

横断プロジェクト優先度履歴ダッシュボード は、複数プロジェクトの優先度、状態、更新履歴を横断して確認する。

## 対象ユーザー

- PM、個人開発者、複数案件を持つチーム

## 主な価値

- 必須項目の不足を自動検出します。
- 手動テスト、導入手順、リリース前確認を同じドキュメント体系で確認できます。
- 競合プロダクトと公式標準を基準に、QCDSを実装とドキュメントの両方で評価します。

## 使い方

```powershell
cd D:\AI\ProjectManagement\cross-project-dashboard-priority-history
npm test
node src/cli.mjs samples/sample-input.json manual-output
```
